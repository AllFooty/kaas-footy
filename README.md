# Kaas Footy

A modern football tournament management system built with Next.js 14, TypeScript, and TailwindCSS.

## Features

- Create and manage football tournaments
- Support for different tournament formats (League, Knockout, Group+Knockout)
- Real-time match updates
- Team management
- Tournament standings and statistics
- Dark mode support
- Modern, responsive UI

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Shadcn UI
- React Query
- Zustand (coming soon)
- Supabase (coming soon)
